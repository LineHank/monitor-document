# 功能
1、统计当前一级告警、二级告警、三级告警数量。

2、统计当前告警列表

3、统计当前恢复列表

# 详情
告警分为 一级告警、 二级告警、 三级告警，重要程度从前往后

告警列表内容包含：主机、发生时间、严重性、指标（具体告警的类型: CPU 内存 硬盘等）、持续时间、告警次数

恢复列表内容包含：主机、发生时间、严重性、指标（具体告警的类型: CPU 内存 硬盘等）、持续时间、告警次数
